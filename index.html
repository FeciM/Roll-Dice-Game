    Something about my first game

    This is my first game I made with the help of a video course

    With this game I tried to practice my acquired knowledge in js.

    The whole game is made in pure javascript with the help of DOM.

    Modal is taken from Code Pen website

    Rules

    The rules are simple.

    The first player starts by rolling the dice.

    The player can roll the dice more than once and the score is added together

    If the score is sufficient, just click on hold the score and the other player can start playing

    The first player to reach a score of 250 wins.

    However, if one roll of the dice is 15, the player loses all points and plays the second player.

    Enjoy the game :)
